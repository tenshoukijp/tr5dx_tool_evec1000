[概要]
  「Evec1000DX」は旧版の太閤立志伝Ⅴにあった「千階堂ツール」を太閤立志伝ⅤDXで動作させるためのツールです。
  作ったばかりでバグだらけかと思うので、バックアップを取ることをお勧めします。また、バグ報告いただけると有難いです。
  旧版に比べ操作方法が若干煩雑になっているので、ご利用の際は「使い方」をよくご確認ください。

[動作環境]
  .Net Framework 4.7.2 以上 (Windows10なら多分標準で入っています)

[使い方]
  (1) 本ツールの「Samples」フォルダから「___千階堂後にこれだけをコンバート.txt」を公式イベコンの「SCRIPT_JP」フォルダにコピーする
  (2) 千階堂記述が含まれたソースファイルの拡張子を「.ev1000」に変更し、公式イベコンの「SCRIPT_JP」フォルダに置く（「Samples」フォルダにいくつか例があります）
      拡張子って何？という方は⇒ https://www.pc-master.jp/sousa/kakuchousi.html
  (3) 「Evec1000DX」を立ち上げる。この時、旧版の千階堂ツールも同時に立ち上がります。旧版の千階堂ツールのメッセージボックスはOKを押しておいてください
  (4) 「Evec1000DX」で公式イベコンの「SCRIPT_JP」フォルダのパスを設定する
  (5) 「Evec1000DX」で「千階堂ソースコード⇒テキストファイル」ボタンを押す（「.ev1000」ファイルが「.txt」ファイルに変換されます）
  (6) 公式イベコンを立ち上げ、(5)で生成したソースコード（.txtファイル）を含めてコンバートを行う
  (7) 「Evec1000DX」で「元の.evmファイル⇒置換後の.evmファイル」ボタンを押す（(6)で生成された「.evm」ファイルの書き換えが行われます）
  (8) 公式イベコンで「___千階堂後にこれだけをコンバート.txt」のみチェックを入れ、コンバートを行う
  (9) 公式イベコンの「OUT_JP」で出来たファイルを「マイドキュメント」にある「EVENT」フォルダに入れる

[セーブナンバーについて]
  DX版では、セーブナンバーは全部で65個までしか保存することができません。
  このうち、0～22のIDは公式で予約されていることが判明しています。
  本ツールでセーブナンバーを使う際は、残りの23～64のIDを割り当てます。（つまり最大42個まで使える）

  セーブナンバーのIDをリセットする際は、オリジナル千階堂の「Evec1000\SvNumId.Dat」ファイルを消してください。
  もう少し細かくID割り当ての設定を変更するためのツールも作りたいと思っていますが、現状は削除で対処してください。

[サンプルについて]
  ・新武将開幕札設定:   ゲーム開始時に全新武将に悪名札以外の札を配布するイベント
  ・ミニゲームで遊ぶ:   酒場に入ると、水路建設、木合わせ、軍学上海のいずれかのミニゲームが遊べるイベント
  ・セーブナンバー金庫: セーブナンバー実験用のイベント。座に入るとセーブナンバーによって実現した金庫に入出金できる
