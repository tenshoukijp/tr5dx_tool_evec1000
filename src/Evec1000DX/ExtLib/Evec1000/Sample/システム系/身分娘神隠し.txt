チャプター:{
	
//イベント:身分娘神隠し
//制作・頴娃烏賊男
//連絡先・
//属性・千階堂方式イベント
//配布条件・改変転載自由
//概要・「セップク」や新武将エディタを使用して、新武将や汎用ライバルの所に身分・
//娘の人物を登録しているエロい人用のイベントです。
//（この方法では主人公以外の嫁を登録したりして、世界観の充足を図ることも可能です。
//吉川元春の嫁やら、明智秀満の嫁やら、姉小路頼綱の嫁やら…閑話休題）
//「セップク」で身分を娘にしただけの場合、酒場の依頼の対象になってしまう可能性が
//あったり（未確認）、新武将の場合道場で剣豪の噂に上ってしまったりします。
//これらの問題は、神隠しフラグを立てることで解決します。
//シナリオ開始時に、汎用ライバル枠でかつ身分が娘の人物の神隠しフラグをONにします。
	イベント:身分娘神隠し{	//イベントの始まり
		属性:一度だけ
		発生契機:ゲームスタート時
		発生条件:{
			//コンテナ設定:(人物,無効,無効)//新武将にも娘がいる場合はこっち
			コンテナ設定:(人物,カテゴリ,汎用ライバル)//そうでないならこっち
			コンテナ絞り込み:(人物,身分,娘)
			コンテナ除外:(人物,出現フラグ,未出現)
			コンテナ除外:(人物,死亡フラグ,死亡)
			
			調査:(コンテナカウンタ)>=(1)
		}
		スクリプト:{//スクリプトの始まり
			ループ:{
				コンテナ選択:(人物Ａ,先頭)
				ブロック脱出:(偽)
				コンテナ選抜:(削る,1)
				ひとりごと:[[＠千階堂＠更新＠人物：：人物Ａ。神隠しフラグ＠1]]
				//更新:(人物Ａ.留守フラグ)(在宅)
				//更新:(人物Ａ.外出禁止フラグ)(真)
				//会話:(人物Ａ,無効)[[神隠しっ！]]
			}
		}
	}
}
